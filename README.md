# CI/CD Demo Environment

This repository contains a simple Todo App for demonstrating CI/CD practices.

## Project Structure

- `todo-app/` - Express.js REST API for managing todos

## Getting Started

1. Navigate to the todo-app directory:
   ```bash
   cd todo-app
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the application:
   ```bash
   npm start
   ```

4. Run tests:
   ```bash
   npm test
   ```

The app will be available at http://localhost:3000
